The story

From capstone project to ChemKit

ChemKit started as my SHS capstone project, originally made as a chemistry guide for incoming Grade 12 students. I recently rebuilt and rebranded it into a cleaner set of browser-based tools for elements, moles, gas laws, pH, and solution concentration.

The toolkit

Six tools. One goal: get you unstuck.

Each one solves a single problem well — no clutter, no sign-up walls, no ads.

Quick guide

Which tool should I use?

  • Need element data? Use Element Explorer.
  • Need moles or particles? Use MoleMate.
  • Need gas law variables? Use GasLab.
  • Need pH, pOH, or ion concentration? Use pH Calculator.
  • Need molarity, dilution, or percent concentration? Use SolutionLab.
  • Need to balance a chemical equation? Use Chemical Equation Balancer.
