# Contributing to ChemKit

Thanks for checking out ChemKit.

ChemKit is a student-built collection of small chemistry tools. Contributions are welcome, especially for bug fixes, calculation corrections, UI improvements, accessibility fixes, and suggestions for new tools.

## Ways to contribute

- Report bugs or incorrect results
- Suggest new chemistry tools
- Improve wording or explanations
- Improve UI and accessibility
- Fix broken links or layout issues

## Reporting issues

If you find a problem, please open a GitHub issue and include:

- What tool or section has the issue
- What you expected to happen
- What actually happened
- Steps to reproduce the issue
- Screenshots, if helpful

For calculator-related issues, include the exact input values and expected answer when possible.

## Pull requests

Small pull requests are welcome.

Before opening a pull request:

- Keep the project static and browser-based
- Do not add a backend, database, authentication, or sign-up system
- Keep the ChemKit visual style consistent
- Make sure existing tool links still work
- Keep changes focused and easy to review

## Notes

ChemKit is mainly a student project and learning project, so suggestions are appreciated even if they are small.
