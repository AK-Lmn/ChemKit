# ChemKit

ChemKit is a modern chemistry mini-tools hub built for students who want quick answers, clean explanations, and interactive learning tools.

It brings together three tools:

- **Element Explorer** — an interactive periodic table with search, filters, favorites, and atomic details.
- **MoleMate** — a mole calculator with mass units, particle estimates, and local calculation history.
- **GasLab** — a guided gas laws calculator for Boyle’s Law, Charles’ Law, Gay-Lussac’s Law, Avogadro’s Law, and the Ideal Gas Law.
