# ChemKit

ChemKit is the main landing hub for a collection of student-friendly chemistry tools built for quick answers, clean explanations, and interactive learning.

The site is a static frontend built with Vite, React, and Tailwind CSS and is ready to deploy on Vercel.

It brings together seven tools:

- **Element Explorer** - an interactive periodic table with search, filters, favorites, and atomic details.
- **MoleMate** - a mole calculator with mass units, particle estimates, and local calculation history.
- **GasLab** - a guided gas laws calculator for Boyle's Law, Charles's Law, Gay-Lussac's Law, Avogadro's Law, and the Ideal Gas Law.
- **pH Calculator** - a live pH, pOH, and ion concentration converter.
- **SolutionLab** - a calculator for molarity, dilution, and percent concentration.
- **Chemical Equation Balancer** - an equation balancer with atom-count checks.
- **Stoichiometry Solver** - guided mole and gram conversions from balanced equations.

## Local development

```bash
npm install
npm run dev
```

Create a production build with `npm run build`.
